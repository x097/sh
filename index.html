Школа.by на iOS

Если вы читаете этот текст, значит вы уже знаете, что приложение Школа.by заблокировано в App Store (и Google Play) из-за жалобы schools.by о якобы нарушении их авторских прав.

Если на смартфон или планшет с Android можно легко установить приложение не из Google Play, то в случае с iPhone или iPad установить приложение не из App Store значительно сложнее. И ключевой момент здесь — цифровая подпись приложения (и используемый для этого сертификат).

Но школьники и их родители должны иметь возможность пользоваться тем мобильным приложением электронного дневника, которое им больше нравится (даже если саму платформу электронного дневника школы выбирают не спросив их согласия).

Поэтому группа небезразличных граждан при поддержке китайских товарищей подготовила данную инструкцию с описанием возможных вариантов установки приложения Школа.by на iPhone или iPad.

Вариант 1. Корпоративный сертификат

Характеристика: Простой в использовании, но не очень надежный.

Суть: Использование приложения, уже подписанного корпоративным сертификатом китайских товарищей.

Преимущества:

  • бесплатно
  • довольно простая процедура установки приложения

Недостатки:

  • сертификат, использованный для подписи, может быть отозван в любой момент, и приложение перестанет работать

Инструкция:

  1. Если у вас iOS 16, включите режим разработчика: «Настройки» — «Конфиденциальность и безопасность» — «Режим разработчика»
  2. Установите приложение Школа.by, пройдя по ссылке: ссылка
  3. В настройках устройства выберите «Основные» — «VPN и управление устройством» (или «Профили и управл. устройством» на старых версиях iOS)
  4. В разделе «Корпоративные приложения» (или «Корпоративная программа») включите доверие для сертификата Hunan Airport Management Group Co. Ltd.

Вариант 2. Подпись собственным бесплатным сертификатом

Характеристика: Для установки приложения требуется компьютер. Процедуру нужно повторять каждую неделю.

Суть: Подпись приложения с помощью специальной программы на компьютере с использованием бесплатного сертификата, доступного для любого Apple ID (но действующего только 7 дней).

Преимущества:

  • бесплатно

Недостатки:

  • для установки приложения требуется компьютер
  • приложение нужно переподписывать и переустанавливать (поверх существующего, данные сохраняются) каждую неделю (иначе перестает запускаться)

Инструкция и подробная

Есть альтернативный вариант, требующий использование компьютера только один раз.

При использовании такого способа скачайте установочный файл приложения Школа.by (.IPA) по ссылке.

Вариант 3. Покупка сертификата

Характеристика: Простой вариант установки, но стоит денег (от 15$).

Суть: Использование для подписи приложения платного сертификата разработчика, купленного на одной из площадок, предлагающих такие услуги.

Преимущества:

  • простая установка
  • подписанное приложение работает год

Недостатки:

  • стоит денег
  • тоже потенциально может быть отозван до истечения года, и приложение перестанет работать

Инструкция

При использовании такого способа скачайте установочный файл приложения Школа.by (.IPA) по ссылке.
